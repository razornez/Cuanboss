# BMAD Planning Index

This folder contains planning artifacts for an AI-powered UMKM Growth Agent SaaS.

Important naming note:
- The SaaS/product name is not finalized yet.
- Avoid using a final product name in core documents until branding research is complete.
- Working description: **AI UMKM Growth Agent OS**.

Important pricing note:
- Pricing is not fixed yet.
- Any number mentioned during ideation is only a sample.
- Final pricing must be calculated from cost structure, margin target, support/onboarding cost, usage limits, and merchant ROI.

## Core Principle

This product should not become just another website builder, CRM, or social media scheduler.

The core promise is to help UMKM owners create and manage sales opportunities every day through visible agent work:

- Daily tasks
- Prospect discovery
- Follow-up guidance
- Website and campaign setup
- Business strategy recommendations
- Capital readiness guidance
- Live reports
- Measurable progress toward sales
