# Cuanboss

Working repository for an AI-powered UMKM Growth Agent SaaS concept.

This repo stores BMAD-style planning documents for a SaaS that helps UMKM create sales opportunities, manage daily growth tasks, track prospects, improve closing, and measure business impact.

> Product name, pricing, and packaging are intentionally not finalized yet.

## Current Focus

- Build an agentic SaaS concept for UMKM growth.
- Help business owners feel they have daily-working Sales, Marketing, Strategy, Capital, and IT agents.
- Avoid becoming only a website builder or social media scheduler.
- Focus on measurable sales opportunity creation, follow-up, conversion, and business readiness.
